# Low-Power INT8 AI Accelerator

## Overview

This project implements a low-power pipelined INT8 AI accelerator using Verilog HDL.

The architecture includes:
- Parallel MAC Array
- Dot Product Engine
- Matrix Multiplication Unit
- Systolic Array Architecture
- FSM-Based Controller
- Clock Gating Optimization
- Performance Benchmarking
- Python-Based Visualization

## Features

- INT8 AI computation
- Pipelined RTL architecture
- Parallel processing
- Low-power clock gating
- Matrix multiplication engine
- Systolic array dataflow
- Performance monitoring
- Python benchmarking tools

## Technologies Used

- Verilog HDL
- Python
- Icarus Verilog
- GTKWave / VCD simulation

## Folder Structure

rtl/         -> RTL source files  
tb/          -> Testbenches  
software/    -> Python benchmarking tools  
sim/         -> Simulation outputs
